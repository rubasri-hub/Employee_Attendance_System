# Employee Attendance System

## Project Description
Employee Attendance System is a mini project developed using Python and MySQL. It helps manage employee records and attendance efficiently.

## Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee
- Search Employee
- Mark Attendance
- View Attendance

## Technologies Used
- Python
- MySQL
- MySQL Connector
- VS Code

## Database
Database Name:
employee_attendance

Tables:
- employee
- attendance

## How to Run
1. Open the project in VS Code.
2. Install MySQL Connector:
   pip install mysql-connector-python
3. Run:
   python main.py
4. Enter your MySQL password when prompted.
